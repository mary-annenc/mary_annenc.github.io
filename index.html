Academic & Personal Projects

April 14, 2026

Menstrual Cycle Training Calculator
This project is a computational tool designed to model the menstrual cycle and explore how hormonal fluctuations may influence training strategies. By imputing cycle length and current day, the calculator determines the users current phase (menstrual, follicular, ovulatory or luteal), and provides evidence-based training recommendations. The project includes programmatically generated visualizations that dynamically update based on user input. These graphs represent hormonal trends and recommended training intensity across the cycle, with the user’s current day highlighted to show real-time positioning within the cycle.
