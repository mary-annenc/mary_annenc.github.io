# Menstrual Cycle Training Calculator
## Project Overview and Purpose
This project is a computational tool that models hormone fluctuations throughout the menstrual cycle and how these changes may affect exercise performance and training strategies. 
Using Python, the model simulates estrogen and progesterone levels and generates a corresponding training intensity profile based on user input.
In addition, a Scratch-based simulation reinforces the underlying programming logic by allowing users to input their cycle day and receive phase-specific training recommendations.
The purpose of this project is to demonstrate how computational tools can be applied to kinesiology, specifically in the context of women’s health. Hormonal fluctuations across the menstrual cycle can impact energy levels, strength, and recovery, yet many training programs do not account for these changes. This project provides a simplified model that visualizes these patterns and translates them into practical training insights.

## Tools and Technologies used
- **Python**(Numpy,Matplotlib)- This was used to model hormone levels and generate the dynamic visualizations.
- **Kaggle Notebook**- This was used for developing and hosting the code and running the analysis.
- **Scratch** - This was used to create a a visual, user-interactive representation of the Python code.
- **Github Pages** - This was used to host the public portfolio website.
- **HTML/CSS** -This was for structuring and presenting the project. 

## How to navigate the project 
**Python Model**
1. Open the linked notebook  
2. Modify the input values:
   - 'cycle_length'
   - 'current_day'  
3. Run the code cells  
4. View the updated graphs showing hormone levels and training intensity  

**Scratch Simulation**
1. Open the embedded Scratch project  
2. Enter a cycle day when prompted  
3. View the resulting phase and training recommendation
